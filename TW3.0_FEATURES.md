# TW3.0 围棋解说引擎 - 统一系统

## 系统概述
TW3.0是围棋解说引擎的第三代产品，整合了之前TW1.0（核心分析）和TW2.0（交互界面）的功能，形成一个完整的围棋AI系统。

## 核心特性

### 1. 统一架构
- 不再区分TW1.0和TW2.0，而是单一完整的系统
- 核心分析与交互界面无缝集成
- 共享数据和服务层

### 2. 实时分析能力
- 用户在界面下棋时，立即调用核心分析引擎
- 实时提供胜率变化
- 即时意图分析
- 实时后续变化预测
- 死活、厚薄、轻重分析

### 3. 模块化设计
- 前端交互模块（原TW2.0功能）
- 后端分析模块（原TW1.0功能）
- 共享服务模块（数据、Katago集成等）

## 功能列表

- 后续变化预测
- 死活判断
- 厚薄分析
- 轻重判断
