# TW围棋解说引擎 - 系统整合计划

## 核心理念
TW引擎是一套完整的围棋AI系统，整合了分析、解说、交互功能于一体：
- TW1.0：核心分析与解说引擎（后台服务）
- TW2.0：交互式界面与实时分析（前端服务）
- 两者共同构成一个完整的围棋AI系统
