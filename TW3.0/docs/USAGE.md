# TW Go Commentary Engine (TW3.0) 使用说明

## 系统要求
- Python 3.7+
- tkinter (通常随Python一起安装)
- 2GB以上可用内存

## 快速开始

### 1. 启动系统
```bash
# 确保有执行权限
chmod +x start_tw3.sh

# 启动系统
./start_tw3.sh
```

### 2. 系统组成
启动后系统包含以下组件：
- 主程序：TW3.0集成系统（GUI界面）
- 学习模块：持续学习进程
- 监控模块：调试监控系统

## 功能介绍

### 主界面功能
- 19×19标准围棋棋盘
- 实时分析结果显示
- 胜率变化曲线
- 意图分析
- 后续变化预测
- 死活判断
- 厚薄分析
- 轻重分析

### 操作说明
- 点击棋盘任意位置落子
- 使用"悔棋"按钮撤销上一步
- 使用"重新分析"按钮刷新分析结果
- 查看右侧分析面板了解详细信息

## 高级功能

### 持续学习
系统会在后台持续运行学习进程，不断优化分析能力。

### 调试监控
系统内置调试监控，确保各组件正常运行。

## 故障排除

### 界面无响应
1. 检查Python环境是否正确安装
2. 确认tkinter库可用
3. 查看debug日志

### 分析结果异常
1. 等待系统初始化完成
2. 尝试重新分析当前局面

## 停止系统
```bash
pkill -f 'python.*\(integrated_system\|continuous_learning\|debug_monitor\)'
```

## 文件说明
- `integrated_system.py`: TW3.0集成系统，包含完整的分析引擎和界面
- `continuous_learning.py`: 持续学习模块，实现自我迭代优化
- `debug_monitor.py`: 调试监控系统，确保系统稳定运行
- `start_tw3.sh`: 一键启动脚本
