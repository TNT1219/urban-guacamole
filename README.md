# TW Go Commentary Engine (TW3.0)

TW Go Commentary Engine 是一个综合性的围棋AI解说系统，整合了TW1.0（核心分析引擎）和TW2.0（交互界面）的功能，形成了一个功能更强大、更完善的围棋AI系统（TW3.0）。

## 项目概述

TW3.0是一个专业级的围棋分析与解说系统，具有以下特点：

- **专业级解说**：提供类似职业棋手的专业解说
- **意图识别**：深度分析棋手的落子意图
- **多维度分析**：包括胜率变化、后续变化、死活判断、厚薄分析、轻重分析等
- **交互界面**：提供直观的用户交互界面
- **自我改进**：具备持续学习和自我优化能力
- **实时分析**：支持实时对弈分析
- **SGF棋谱支持**：可导入、分析和导航SGF格式的棋谱
- **报告导出**：可导出详细的分析报告

## 环境要求

- Python 3.7+
- tkinter (通常随Python一起安装)
- 2GB以上可用内存

## 安装依赖

```bash
pip install -r requirements.txt
```

主要依赖：
- numpy: 数值计算
- Pillow: 图像处理（可选）
- python-chess: 棋类游戏处理（可选）
- sgfmill: SGF棋谱解析

## 快速启动

```bash
# 给启动脚本授权
chmod +x start_tw3.sh

# Linux/Mac启动系统
./start_tw3.sh

# Windows启动系统
start_tw3.bat
```

## 系统组成

- **TW1.0**: 核心分析引擎，负责围棋分析和解说生成
- **TW2.0**: 交互界面，提供用户友好的操作界面
- **TW3.0**: 集成系统，整合了TW1.0和TW2.0的功能，并增强了以下功能：
  - SGF棋谱导入与分析
  - 棋局导航（前进/后退）
  - 分析报告导出
  - 更精准的多维度分析

## 功能特性

1. **实时分析**：用户在界面下棋时，系统实时提供分析
2. **意图识别**：深度分析每一步棋的意图
3. **多维度评估**：提供胜率、变化、死活、厚薄、轻重等多方面分析
4. **自我学习**：系统具备持续学习和改进的能力
5. **可视化界面**：直观展示棋局和分析结果
6. **SGF支持**：可导入和分析SGF格式的棋谱文件
7. **棋局导航**：支持在棋谱中前进、后退查看不同局面
8. **报告导出**：可导出详细的分析报告
